Oppdaget HTML på barneskolen og holdt på med dette i flere år. På videregående var jeg med på mitt første prosjekt med PHP og MySQL, men hovedsaklig som designer.

Har siden den tid fått interesse for oppsett av servere, hvor det startet med Windows Server 2000, og har utviklet seg til eksperimentering med proxmox og mer linux relaterte operativsystemer som for eksempel ubuntu og debian.

Videreutviklet ferdighetene mine i PHP og MySQL på Høgskolen i Oslo og Akershus, hvor interessen for JavaScript/jQuery også har meldt seg. Har også lært litt om andre programmeringsspråk for eksempel Java, ASP.NET og C#.

Ellers har jeg nylig kjøpt meg ny sykkel, for å ta opp igjen sykkel som transportmiddel, når jeg syklet mye tidligere.

Du kan også se meg online på steam, hvor det som oftest går i Dota 2 sammen med venner og bekjente. Eller du kan finne meg online på minecraft, selv om det blir mindre og mindre spilling nå om dagen.
